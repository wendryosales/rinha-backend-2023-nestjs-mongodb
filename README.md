Rinha de Backend NestJS & Mongodb

## Instalação

```bash
$ npm install
```

## Rodando o app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

###  Use `docker-compose up` para subir todos containers docker
